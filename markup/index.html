Query2JSON

What is Query2JSON?

Query2JSON solves a very simply problem and is a quick and easy way. Several services (i.e. Twilio) use RESTlike webhooks for reacting to incoming data. These services make requests like this.
http://www.yourwebsite.com/callback?cat=hat&fish=stick
This works fine if you're writing a custom application, but sometimes you need to "glue" the data from this service into another service or database host (i.e. Firebase). Some of these services expect requests like this.
http://www.secondservice.com/youraccount/
-> {"cat": "hat", "fish":"stick"}
Query2JSON solves this little incompatibility by providing a way to translate the request without running your own server. A request with Query2JSON works like this.
http://www.query2json.com/{ID}?cat=hat&fish=stick
[Query2JSON Translates Request]
http://www.secondservice.com/youraccount/
-> {"cat": "hat", "fish":"stick"}
